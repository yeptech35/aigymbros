# 🎯 AIGYMBROS - AI Gym Coach App

**PT Marhaen AI** - Your Personal AI Fitness Coach!

## 📋 Features

✅ **Home Dashboard**
- Daily budget tracking (Bajet Harian)
- Current weight monitoring
- Streak counter (Motivation!)
- AI Coach Chat (Coach Mat)
- Daily meal planning
- Progress statistics

✅ **Diet Tracking**
- Complete meal breakdown
- Calorie counting
- Meal timing (Sarapan, Makan Tengahari, Makan Malam)
- Budget-friendly recipes
- Daily calorie progress

✅ **Workout Program**
- 5-day split routine:
  - Monday: Chest & Triceps
  - Wednesday: Back & Biceps
  - Friday: Legs
  - Daily Cardio
- Set and rep recommendations
- Workout duration tracking

✅ **Profile**
- User statistics
- BMI calculation
- Total workouts logged
- Active days counter
- Settings & preferences

## 🚀 Quick Start

1. Visit: `https://yeptech35.github.io/aigymbros`
2. Start tracking your fitness journey
3. Data is saved locally in your browser
